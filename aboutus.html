About This Research Project

A comprehensive academic study examining television energy consumption patterns in the Australian consumer electronics market

Project Overview

This research project presents a systematic analysis of television energy consumption trends in Australia, developed as part of COS30045 - Data Visualisation coursework. The study employs modern data visualisation techniques to communicate insights about energy efficiency patterns across different television technologies, brands, and screen sizes available in the Australian market.

Academic Context

Developed by Dong Nguyen Gia Huy (Student ID: 104993450) for Swinburne University of Technology's Data Visualisation unit. This project demonstrates proficiency in data analysis, web development, and academic communication standards. The research addresses contemporary concerns about household energy consumption and provides evidence-based insights for consumer decision-making.

Research Significance

As Australian households face increasing energy costs and environmental sustainability challenges, understanding appliance energy consumption becomes crucial for informed consumer choices. This research contributes to the broader discourse on energy efficiency by providing accessible, data-driven insights that can influence purchasing decisions and promote sustainable consumption patterns.

Technical Implementation

Development Framework

Built using modern web technologies including HTML5, CSS3, and vanilla JavaScript. The responsive design ensures accessibility across desktop and mobile devices, with WCAG 2.1 compliance for inclusive user experience. Enhanced with GitHub Copilot assistance for code optimization and accessibility features.

Data Visualisation Approach

Static visualisations generated using industry-standard tools and integrated as optimised image assets. The approach prioritises loading performance while maintaining visual clarity and academic presentation standards suitable for data communication and analysis.

Academic Standards

The project adheres to academic writing conventions with proper citations, methodology transparency, and ethical data usage guidelines. All content is presented in accordance with university standards for data visualisation projects and academic portfolio development.
